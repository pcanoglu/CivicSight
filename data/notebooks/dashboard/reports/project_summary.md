# CivicSight Project Summary

CivicSight is a nonprofit analytics initiative designed to help organizations improve donor retention and fundraising decisions.

## Current Version
This first version uses sample donor data to identify donor behavior patterns and at-risk donors.

## Key Outputs
- Donor data table
- Total donation amount
- Average donation amount
- Predicted donor retention rate
- At-risk donor list
- Recommended re-engagement priorities

## Intended Users
Small and mid-sized nonprofit organizations that lack access to advanced analytics tools.

## Long-Term Goal
To build scalable, low-cost analytics infrastructure that strengthens nonprofit financial sustainability across the United States.
