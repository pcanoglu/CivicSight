# CivicSight

CivicSight is a data-driven platform that helps nonprofit organizations improve donor retention, optimize fundraising strategies, and strengthen financial sustainability through analytics.

## Problem
Nonprofits often lack access to data tools, making it difficult to understand donor behavior and maintain consistent funding.

## Solution
CivicSight provides:
- Donor retention predictions  
- At-risk donor identification  
- Data-driven fundraising insights  

## Vision
To build scalable analytics tools that improve nonprofit efficiency across the United States.
